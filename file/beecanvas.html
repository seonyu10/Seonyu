INTRO

BEECANVAS

BeeCanvas is a visual collaboration tool that lets you share and comment a image, links, or other multimedia resources on a canvas whatever you want.

USE CASE

Visual workspace for creative people

BeeCanvas is designed for visual projects, which serves as a whiteboard for graphic and visually driven projects. Itâ€™s like Google docs for designers, artists, and those working on visual projects. It has to be easy to use.

DESKTOP INTERFACE

Consistent User Interface and Experience

BeeCanvas is basically desktop first software but it also has to be easy to use in mobile and tablet as well. I focused on the conherent UI design and UX in any devices.

01

PROJECT-BASED WORKSPACE

BeeCanvas is a project-based workspace. Members of the working group can join and be invited into each of the projects. The cover image of projects can be selected individually for distinction.

02

A VIRTUAL WORKSPACE

The main purpose is to bring the workspace into a full screen canvas. Through the organisation of each collaborator's activities and tasks, a smooth work flow is provided for each project. The project interface gives a sense of a virtual working space for the users.

03

VIRTUAL WHITEBOARD

The tool bar has functions of a drawing tool, post-its, links embed, file upload integration with google docs and dropbox. The slide navigator resembles a similar UI structure as MS "Power Point", providing a user friendly interface. Members can collaborate and give real-time feedbacks and comments.

MOBILE/TABLET DESIGN

Virtual workspace on your hand

I designed BeeCanvas mobile friendly, mainly because it is extremely important users feel the same experience on mobile with desktop and see the changes and modify or collaborate with teammates easily.

MOBILE INTERACTION

Canvas

Easy to view and edit the canvas in mobile. Adding a Post-its has a fun and intuitive interaction that you just need to drag and drop with one finger.
